# Red Carpet Goban

Ray-traced 3D Go board with GLSL rendering.

## Easy Setup

For ready-to-use bundles with pre-configured engines and all required assets, visit [hraj.si/goban](https://hraj.si/goban)

## Advanced: Bare Executables

These are standalone executables for users who want to set up their own environment.

**Requirements:**
- Configuration files, shaders, fonts (Noto Sans), sounds from a working installation or build
- A GTP-compliant engine (GNU Go, KataGo, or similar)
- Engine paths configured in `config/<language>.json`

**Available builds:**
- **Windows x64** - Recommended for 64-bit Windows
- **Windows x86** - For 32-bit Windows
- **Linux x64** - For 64-bit Linux
- **macOS x64** - Experimental

Both Windows builds should work with any configured GTP-compliant engine.
