  Display Adjustments:
  ┌─────┬──────────────────────────┐
  │ Key │          Action          │
  ├─────┼──────────────────────────┤
  │ ]   │ Increase gamma           │
  ├─────┼──────────────────────────┤
  │ [   │ Decrease gamma           │
  ├─────┼──────────────────────────┤
  │ =   │ Increase contrast        │
  ├─────┼──────────────────────────┤
  │ -   │ Decrease contrast        │
  ├─────┼──────────────────────────┤
  │ 0   │ Reset contrast and gamma │
  └─────┴──────────────────────────┘
  SGF Navigation (when a game is loaded):
  ┌───────────────┬───────────────┐
  │      Key      │    Action     │
  ├───────────────┼───────────────┤
  │ → / Space     │ Next move     │
  ├───────────────┼───────────────┤
  │ ← / Backspace │ Previous move │
  └───────────────┴───────────────┘
